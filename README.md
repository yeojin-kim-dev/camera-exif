# Camera EXIF Extraction (PoC)

본 저장소는  
사진 파일에 포함된 EXIF 메타데이터(ISO, 셔터속도)를 자동으로 추출하고,  
장면별 촬영 설정값을 분석하기 위한 사전 검증(PoC) 코드입니다.

AI 기반 촬영 설정 추천을 목표로 하며,  
그에 앞서 실제 촬영 데이터로부터 전문가 설정값 기준 데이터셋을 구축할 수 있는지 확인하는 단계입니다.

---

## 프로젝트 배경 및 목적

- 사진 파일에는 촬영 시 사용된 ISO, 셔터속도 등의 설정값이 EXIF 형태로 저장됨
- 이 설정값들을 코드로 안정적으로 추출할 수 있는지 기술적으로 검증
- 장면(scene)별 실제 설정값 분포를 분석하여
  향후 전문가 기준 설정값 정의에 활용

---

## 주요 스크립트 설명

### check_exif.py
- 사진 파일에 EXIF 메타데이터가 존재하는지 확인
- ISO, 셔터속도(ExposureTime) 태그 추출 가능 여부 검증
- EXIF 파싱의 기술적 가능성 확인 목적

### exif_to_excel.py
- 장면별로 분류된 사진 폴더 순회
- EXIF에서 ISO, 셔터속도 자동 추출
- 셔터속도를 초 단위(float)로 변환
- 분석용 엑셀 데이터셋 생성

- AI 촬영 설정 추천 결과와 비교 평가에 활용 예정
